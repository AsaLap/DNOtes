Opéron des [[Bactérie lactique|bactéries lactiques]] qui code pour :
- CitR : régulateur de transcription induit par le [[Acide citrique|citrate]] lui-même.
- MaeP : [[Citrate perméase]] permet de faire entrer le citrate dans la cellule.
- [[Citrate lyase]] : permet de catalyser la transformation de [[Acide citrique|citrate]] en [[Oxaloacétate|oxaloacétate]] avec perte d'un [[Acide acétique|acétate]].
- [[Oxaloacétate décarboxylase]] : permet la [[Décarboxylation|décarboxylation]] de l'[[Oxaloacétate|oxaloacétate]] en [[Pyruvate|pyruvate]].
