# Notes de révisions pour le DNO

# HOW TO USE:
1. Téléchargez tous les fichiers de ce répertoire (en haut à droite en vert <Code>, sélectionnez "Download ZIP" tout en bas du menu).
2. Décompressez le dossier en gardant le dossier "DNOtes" comme dossier parent.
3. Téléchargez [Obsidian.md](https://obsidian.md/)
4. Ouvrez Obsidian.md
5. Sélectionnez "Ouvrir un dossier comme coffre"
6. Sélectionnez le dossier parent "DNOtes" avec Obsidian
7. Enjoy (naviguer dans les fichiers, filtrez par tags, ouvrez la vue graphique et baladez-vous dedans en filtrant sur certains tags par exemple)

# Public
Ces fiches sont à destination des DNO pour réviser et/ou servir de banque d'information sur des données utiles au métier.
